# Resumo da Solução

### Enfrentamos um desafio com a nova plataforma do Governo em relação ao download de arquivos XML, devido ao alto volume (184 arquivos apenas em janeiro) e à ausência de uma funcionalidade de "baixar tudo" ou "selecionar todos" (até 10 de fevereiro de 2026).

### Para resolver isso, explorei alternativas e bibliotecas para automatizar o fluxo de trabalho usando Python, implementando soluções para autenticação, extração de XML e paginação.

# Solution summary

#### We encountered a challenge with the new GOV platform regarding XML downloads due to the high volume (184 files in January alone) and the absence of a 'download all' or 'select all' feature (as of Feb 10, 2026). 
#### To address this, I explored alternatives and libraries to automate the workflow using Python, implementing solutions for authentication, XML extraction, and pagination.

#### NOTE: To use the software with its current logic, you must place the password and certificate .pem files in the same folder as the application. Even if you compile it into an .exe, both files must remain in the same directory as the executable.
